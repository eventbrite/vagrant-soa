# vagrant-soa

A [Vagrant](http://www.vagrantup.com/) plugin to handle our SOA infrastructre within vagrant.

## Installation

``` bash
vagrant plugin install vagrant-soa
```

## Usage

## Development

``` bash
$ bundle
$ bundle exec vagrant up
```
